Time Series Storage Service based (deprecated)

A time series storage using InfluxDB. This service is deprecated, please use the new TSDB service instead. (http://docs.exosite.com/murano/services/tsdb)

Operations

Storage

  • Timeseries.command() - Executes a command against the database
  • Timeseries.query() - Query the database
  • Timeseries.write() - Writes data into database

Operations

Storage

command

Executes a command against the database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
q string InfluxQL query. Find more information about the format on https://docs.influxdata.com/influxdb/v1.0/query_language
epoch string Timestamps in Unix epoch format
chunk_size string Explicitly set the batch size
Responses
Influx database response - object
Name Type Description
error string InfluxDB error description.
results [ object ] InfluxDB result list.
Operation error details - object
Name Type Description
message string Error message
status_code string Error code

query

Query the database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
q string InfluxQL query. Find more information about the format on https://docs.influxdata.com/influxdb/v1.0/query_language
epoch string Timestamps in Unix epoch format
chunk_size string Explicitly set the batch size
Responses
Influx database response - object
Name Type Description
error string InfluxDB error description.
results [ object ] InfluxDB result list.
Operation error details - object
Name Type Description
message string Error message
status_code string Error code
Example
local query = Timeseries.query("select * from measurement limit 10") local error = query.error local results = query.results

write

Writes data into database

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
body object Object data to write.
body.query string InfluxQL query. Find more information about the format on https://docs.influxdata.com/influxdb/v1.0/write_protocols/write_syntax
Responses
Query Error - object
Name Type Description
error string InfluxDB error description.
Data successfully inserted - nil

Data successfully inserted

Operation error details - object
Name Type Description
message string Error message
status_code string Error code
Example
local query = Timeseries.write("cpu_load_short,host=server02,region=us-west value=0.55 1422568543702900257")
