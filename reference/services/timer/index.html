Timer Service API

Set up timers which will fire events once, on an interval, or on a schedule.

Operations

  • Timer.cancel() - cancel an active timer
  • Timer.cancelAll() - cancel all active timers for a given solution
  • Timer.schedule() - create a scheduled timer
  • Timer.sendAfter() - create a one-shot timer
  • Timer.sendInterval() - create an interval timer

Events

  • timer - timer trigger

Operations

cancel

This operation will cancel an active timer.

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
timer_id string The ID of the timer to cancel
Responses
Operation successful - object
Name Type Description
result string Okay if successful
Operation error information - object
Name Type Description
type string The error type
message string The error description

cancelAll

This operation will cancel all active timers in the given solution.

Responses
Operation successful - object
Name Type Description
result string Okay if successful
Operation error information - object
Name Type Description
type string The error type
message string The error description

schedule

Sets up a timer which will fire on a schedule which is based on the date and time on which the schedule is created.

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
timer_spec object The timer specification
timer_spec.message string, object The message to send when the timer is hit
timer_spec.duration integer The duration of the timer in milliseconds.
Required for send_after and send_interval,
ignored for scheduled tasks.
timer_spec.schedule object The task schedule definition
timer_spec.daily boolean This task should occur daily
timer_spec.hourly boolean This task should occur hourly
timer_spec.weekly boolean This task should occur weekly
timer_spec.yearly boolean This task should occur yearly
timer_spec.monthly boolean This task should occur monthly
timer_spec.timer_id string The ID of the timer. If null, it will be generated for you.
Responses
New timer task information - object
Name Type Description
result string A message describing the result of the operation
timer_id string The ID of the timer
Operation error information - object
Name Type Description
type string The error type
message string The error description
