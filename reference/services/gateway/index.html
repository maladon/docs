Operations

configure

Updates a product identified by {pid}

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
pid ^[a-zA-Z0-9_]+$ Product ID
product object The product object to patch
product.name string The name of this product
product.protocol object Protocols allowed for this product
product.name "cbor", "mqtt", "onep" The name of the protocol
product.dev_mode boolean Is this in dev mode
product.auth_type "certificate", "keypair", "password", "secret", "cik" Authentication type
product.resources object The resources of this product
product.description string The description of this product
product.provisioning object Provisioning means creating a device identity and/or providing authentication objects to devices.
product.enabled boolean Is provisioning enabled for this product. If it is disabled then the device can only communicate if Murano already knows about this device and it has authentication objects.
product.ip_whitelisting object The whitelist IP settings
product.allowed [ string ] An array of IP addresses
product.enabled boolean Is whitelist IPs enabled
product.generate_identity boolean Should the server generate identity for a device of this product. If a connected device does not have a name the server can generate one.
product.presenter_identity boolean Should the server accept identities presented by the device
product.identity_format object Describes the format of the identities (e.g) devices that can be added for a product. The identity format will be checked when the device communicates with murano; if it does not match the product's identity format it will be rejected. When a device is generated, the name will be generated using this format
product.type "mac:48", "mac-48", "mac.48", "uuid", "uuidv4", "base10", "base16", "opaque" The identity type that can be selected from the options below
product.prefix string A prefix for the identity name
product.options object Options for identity format
product.casing "lower", "upper", "mixed" Describes whether the identity should be lower or upper case
product.length number The length of the identity in bytes (base10 and base16 only)
Responses
Product updated successfully. - nil

Product updated successfully.

The error object - object
Name Type Description
code integer(int32) The error code for this error
message string The error message for this error
The error object - object
Name Type Description
code integer(int32) The error code for this error
message string The error message for this error

Name Type Description
auth object The device authentication object
auth.key string The authentication defined by type

list

Returns a list of devices

Arguments
parameters - object - Object containing service call parameters.
Name Type Description
pid ^[a-zA-Z0-9_]+$ Product ID
limit integer Return maximum {limit} devices
before number Return devices modified before this date, used to go through several pages of devices
status string A comma separated value list of status to return any of [whitelisted, provisioned, locked, devmode, reprovision]
identity string A pattern match string for device names describing the which devices should be returned
ipaddress string A pattern match string for the IP
version string A pattern match string for the version
Responses
The device array that matches the query - object
Name Type Description
devices [ object ] List of devices
devices[].auth object The device authentication object
devices[].key string The authentication defined by type
devices[].expire string(date) When does this expire
devices[].state object The device state
devices[].lastip string The timestamp when this device was last seen
devices[].locked boolean Is this device locked. Only not locked devices can communicate with Murano.
devices[].status "provisioned", "whitelisted" The device status; this is an aggregate field"
devices[].devmode boolean Allow streaming of low level logs
devices[].identity string The identity of this device
devices[].lastseen number The timestamp when this device was last seen
devices[].context_id string The pegasus context id
devices[].reprovision boolean Does this device need to be reprovisioned
mayLoadMore boolean Indicate the list is not complete.
The error object - object
Name Type Description
code integer(int32) The error code for this error
message string The error message for this error
